# ContextClip – AI Research Assistant

ContextClip is an AI-powered research assistant that helps users analyze PDFs and URLs to generate structured summaries, key insights, and interactive Q&A.

---

## ✨ Features

* 📄 Upload PDF documents (automatic text extraction)
* 🔗 Add URLs as research sources
* ⚡ AI-powered research synthesis
* 📊 Executive summary and key insights
* 💬 Interactive Q&A chat system
* 🎨 Modern responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Custom CSS

---


## 🚀 Getting Started

### 1. Clone the Repository

```
git clone https://github.com/YOUR_USERNAME/contextclip-ai.git
cd contextclip-ai
```

### 2. Install Dependencies

```
npm install
```

### 3. Start Development Server

```
npm start
```


## ⚠️ Important Notes

* This project uses AI APIs which may require billing
* Frontend API calls are not fully secure
* For production, use a backend (Node.js / serverless functions)
